lateBind

  • lateBind(instance: __type, method: string, ...suppliedArgs: any[]): function
    • Defined in lang.ts:223

    Returns a function that binds a method to the specified object at runtime. This is similar to Function.prototype.bind, but instead of a function it takes the name of a method on an object. As a result, the function returned by lateBind will always call the function currently assigned to the specified property on the object as of the moment the function it returns is called.

    Parameters

    • instance: __type

      The context object

    • method: string

      The name of the method on the context object to bind to itself

    • Rest ...suppliedArgs: any[]

      An optional array of values to prepend to the instance[method] arguments list

    Returns function
